### React application starter based on webpack without using CRA

### Loaders

- babel-loader
- css-loader
- file-loader
- img-optimize-loader
- post-loader
- react-hot-loader
- sass-loader
- style-loader
- ts-loader

### Getting started

1. Intall dependencies

```
npm install
```

2. Run the application

```
npm start
```

3. Build the application

```
npm run build
```
